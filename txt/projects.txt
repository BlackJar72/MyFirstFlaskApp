Desktop Applications

Digit Span Tester:  A small Java application designed to perform automated tests of human working memory.  This was intended as a research tool for the fields of cognitive science and neuroscience; for example, to stimulate working memory and attention while recording brain wave activity for quantitative electroencephalographic analysis.

QflashCards (may be renamed due unintended use of an existing name): A small C++ / Qt application intended as a study tool.

ProcGenLab: A Java Application for experimenting and prototyping in procedural content generation, especially of location maps and pixel / voxel graphics.

Games

Macy Mae's Eating Adventure: An arcade maze chase (or “cop and robbers”) game.  Play as a cat (Macy Mae) trying to eat all the kibbles, salmon, birds, and rodents while being chase by will-o-wisps.

Ship Swarm Shoot-Up: A small, simple arcade space shooter.

Mods

Doomlike Dungeons: A procedural dungeon generator which create huge multiroom dungeons in the style of retro first person shooter levels, and builds them into Minecraft.

Climatic Biomes: A an alternate biome provider written from scratch for Minecraft.  Rather than relying heavily and random noise fields, it uses fields of attraction to areas of land/water, hot/cold, and wet dry.  Large islands (“continents”) and generated based and landiness, then realistic biomes classifications are determined from climate data and game “biomes” are selected based on those.

Mob Drop Stoper: A very small Bukkit plugin which eliminates or replaces items dropped by mobs, originally created to cover up a bug in Cauldron.

Ancient History:

Htmlpp:  Supposedly an “html pre-processor” (no relation to php), this was a very small, simple commandline utility that acted as macro / templating system for off-line generation of html files.
